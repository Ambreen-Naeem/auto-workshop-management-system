# Auto Workshop Management System

A console-based C++ project developed for Programming Fundamentals.

## Project Description

This project simulates an auto workshop management system where workshop operations and customer-related tasks can be managed efficiently through a menu-driven program.

## Features

- Add customer details
- Manage workshop records
- Display information
- Simple menu-driven interface
- data management 

## Technologies Used

- C++
- Object-Oriented Programming

## How to Run

1. Open the project in any C++ IDE
   - Dev C++
   - Visual Studio

2. Compile the file:
   Auto Workshop.cpp

3. Run the program
